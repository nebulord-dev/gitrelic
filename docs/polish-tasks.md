# Polish Tasks

These are some ideas that I want to go over when reviewing and forensically fine-tooth combing each analyzer.  The goal is that at the end of each analyzer session we have a pristine, well-thought out, accurate and valuable display of information for the user.

- Walk through the analyzer and make sure the bottom section reflects accurate and valuable information that isn't just duplicating the hero graph
- Determine if the bottom section needs to interact with the hero graph. 
- If it is a table on the bottom, determine if multiple tabs should be used.  
- Go through each hero graph and determine if it is valuable or not. 
- Determine if a hero graph would be a great addition or replacement to better reflect the data.
- Once the analyzer is refined, add the docs for that analyzer to the VitePress site. (use the Churn docs as an example if needed)
